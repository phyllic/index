A Decentralized Network: Everyone's Invited!

TibCoin runs on a cool peer-to-peer network with simple, awesome rules.

  1. New transactions zoom across to all nodes.
  2. Each node gathers new transactions into a block.
  3. Every node tries to find the super-hard proof-of-work for its block.
  4. Found it! A node broadcasts its awesome block to everyone.
  5. Nodes only accept blocks if all transactions are legit and not spent already.
  6. Nodes show their approval by building the *next* block on top of the accepted one!
